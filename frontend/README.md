# Zuno Frontend

Zuno is the student-facing frontend for the Bihar Board Class 10 Science AI Tutor backend.

## Commands

Install dependencies:

```bash
npm install
```

Start the frontend:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Environment

Create `.env` from `.env.example` if the backend runs on a different URL.

```text
VITE_API_BASE_URL=http://localhost:5000
```
